

# 📊 Smart Attendance System

A **Smart Attendance Management System** built using modern web technologies.
This application helps schools, colleges, or organizations **manage and track attendance efficiently** through a simple and user-friendly interface.

## ✨ Features

* Secure **Admin Login**
* **Attendance Management Dashboard**
* Easy **Attendance Tracking**
* Clean and **Responsive User Interface**
* Fast performance using **Vite**
* Lightweight **SQLite Database**
* Simple and efficient attendance management

## 🛠️ Technologies Used

| Technology     | Purpose                         |
| -------------- | ------------------------------- |
| **React**      | Frontend User Interface         |
| **Vite**       | Fast development build tool     |
| **Node.js**    | Backend server                  |
| **TypeScript** | Type-safe development           |
| **SQLite**     | Database for attendance records |

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Gobiga13/Smart_Attendence_System.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd Smart_Attendence_System
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Development Server

```bash
npm run dev
```

The application will start at:

```
http://localhost:3000
```
